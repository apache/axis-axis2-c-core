2 Unit Tests

To add unit tests for a module there are two steps to be follwed, first add the unit test to the module's test suite, second call it from the main unit test suite.

2.1 Adding a Unit Test to a Module's Unit Test Sutie

Suppose you need to add a new test in the common module

1. Write the test functions in modules/common/test/common_test.c

2. Add the test into the common module's unit test suite

e.g.

CuSuite* axis2_commonGetSuite() {

CuSuite* suite = CuSuiteNew();

SUITE_ADD_TEST(suite, Testaxis2_stream_ops_read);

SUITE_ADD_TEST(suite, Testaxis2_stream_ops_write);

SUITE_ADD_TEST(suite, Testaxis2_log_ops_write);

return suite;

}

2.2 Calling the Unit Test from Main Unit Test Suite

There is a main test suite to call all the unit tests of all the modules at once in modules/test/unit folder

To add a unit test of a module to the main test suite add a 'CuSuiteAddSuite' line to the modules/test/unit/main.c file there.

e.g.

CuSuiteAddSuite(suite, axis2_commonGetSuite());
